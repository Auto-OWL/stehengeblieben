API für Händler

Automatischer Datenaustausch zwischen Ihrem DMS-System und stehengeblieben.de. Unterstützt: Autrado, Audaris, Loco-Soft und alle Systeme mit CSV-Export.

Inhaltsverzeichnis

Einstieg Authentifizierung CSV-Import REST Endpunkte Datenfelder DMS-Systeme Fehlercodes

Einstieg

Die stehengeblieben.de API ermöglicht es Händlern, Fahrzeugdaten automatisch aus ihrem DMS-System zu importieren. Der einfachste Weg ist der CSV-Import — kein Code nötig. Für vollständige Automatisierung steht die REST API zur Verfügung.

upload_fileCSV-Import

Datei aus DMS exportieren, hochladen — fertig. Funktioniert mit jedem System.

apiREST API

Vollautomatischer Sync direkt aus Ihrem System. Erfordert API-Key.

info API-Keys erhalten Sie nach Ihrer Registrierung als Händler.

Authentifizierung

Alle API-Anfragen müssen mit einem API-Key authentifiziert werden. Übergeben Sie den Key im HTTP-Header:

POST https://stehengeblieben.de/api/v1/vehicles
Authorization: Bearer IHR_API_KEY
Content-Type: application/json
warning

Geben Sie Ihren API-Key niemals in Frontend-Code oder öffentlich zugänglichen Dateien weiter.

CSV-Import (empfohlen)

Der CSV-Import ist der schnellste Weg für Händler ohne technische Integration. Das Format entspricht dem Mobile.de Extended Format und ist mit allen gängigen DMS-Systemen kompatibel.

Endpunkt

POST https://stehengeblieben.de/api/import-csv
Authorization: Bearer IHR_API_KEY
Content-Type: multipart/form-data

Feld: file (CSV-Datei, max. 10 MB)

CSV-Spalten (Pflichtfelder)

SpalteTypBeispiel
makeStringVolkswagen
modelStringGolf GTI
priceInteger (EUR)31900
list_priceInteger (EUR)42500
first_registrationDate (YYYY-MM-DD)2024-01-15
fuel_typeEnumPETROL / DIESEL / ELECTRIC / HYBRID
mileageInteger (km)12

Beispiel-Response

{
  "success": true,
  "imported": 8,
  "skipped": 1,
  "errors": [
    { "row": 4, "field": "price", "message": "Ungültiger Wert" }
  ]
}

DMS-Systeme

So exportieren Sie Daten aus den gängigsten Systemen:

Autrado

Inventar → Export → Mobile.de CSV (Extended Format) → Datei auf stehengeblieben.de hochladen

Audaris

Fahrzeugverwaltung → Exporte → Standardformat Mobile.de → CSV herunterladen → importieren

Loco-Soft

Modul Fahrzeuge → Exportieren → Format: Mobile.de v2 → speichern und hier hochladen

Andere Systeme

Alle Systeme die einen CSV-Export im Mobile.de-Format unterstützen funktionieren direkt.
